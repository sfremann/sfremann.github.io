Dialogue tree plugin for Unreal Engine

🚧 In progress UE5 C++ Python

Plugin adding a dialogue tree editor in Unreal Engine 5 for future personal game projects. The dialogue tree asset is made of custom nodes using story flags to check or set, text, audio, animation and camera position. The plugin also handles how dialogue is supposed to play in-game when speaking with characters.


Highlights:

  • Dialogue tree editor.
  • In-game dialogue tree asset behaviour.
  • Additional ProjectSettings to create, save and load custom story flags.

VR application - Haptics

Released in June 2024 Team of 2 UE5 C++ Python

Research project on haptics and how they can improve customer experience when buying products online. An application was developed to provide an experience for participants to try in a user study. I was the only developer on this project at the Digital Future Lab (previously EDM, Expertise Centre for Digital Media), a research centre of Hasselt University.


My Contributions:

  • Changes in TouchDIVER Haptic Gloves API to send simple messages to glove from Unreal Engine.
  • Configurable experience with different play modes (3D, VR), interactions and haptic feedback.
  • Automation tool to launch the experience with specific settings, record player data and save them in a result folder.
